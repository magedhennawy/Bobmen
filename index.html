Bobboard Documentation

local setup

cd ng2angle: move into the projects main directory
npm install: install all the dependencies via NPM
npm start: build the front end and start the server - connect @ https://localhost:3000

Users

Create a User

  • description: create a new user
  • request: PUT /user
    • content-type: application/json
    • body: Object
      • username: String
      • password: String
  • response: 200 | 500 if server error | 409 if user already exists
    • content-type: application/json
    • body: Object
      • username: the username of new user

User Signin

  • description: authenticate user
  • request: POST /user/signin
    • content-type: application/json
    • body: Object
      • username: String
      • password: String
  • response: 200 | 500 server error | 400 bad request | 401 unauthorized
    • content-type: application/json
    • body: Object
      • username: the username of user

User Signout

  • description: signout existing user
  • request: GET /user/signout
    • content-type: application/json
    • query: none
  • response: 200 | 500 if server error
    • content-type: application/json
    • body: Object
      • message: signout message

Password Saver

Get Passwords

  • description: get a user's passwords
  • request: GET /passwordsaver
    • content-type: application/json
    • body: none
  • response: 200 | 500 server error
    • content-type: application/json
    • body: user's passwords

Add Password

  • description: add a user's password to database
  • request: POST /passwordsaver
    • content-type: application/json
    • body: Object
      • title: Website
      • description: password
  • response: 200 | 500 server error
    • content-type: application/json
    • body: Array of Password Objects
      • title: password
      • description: description for the password

Edit Password

  • description: edit a user's password in the database
  • request: PATCH /passwordsaver
    • content-type: application/json
    • body: Object
      • index: index of password object
      • title: new Website
      • description: new password
  • response: 200 | 500 server error
    • content-type: application/json
    • body: Object
      • message: successfull operation message

Delete Password

  • description: delete a user's password in the database
  • request: DELETE /passwordsaver/:index
    • content-type: application/json
    • query: Object
      • index: index of password
  • response: 200 | 500 server error
    • content-type: application/json
    • body: Object
      • message: successfull operation message

  • description: get a list of cities
  • request: GET /weather/cityList
    • content-type: application/json
    • query: none
  • response: 200 | 500 server error
    • content-type: application/json
    • body: city list

Update User's Weather Settings

  • description: get a user's tweets
  • request: POST /weather/settings
    • content-type: application/json
    • body: Object
      • city: String; city name
      • country: String; country 2 letter abbreviation
      • celsius: Boolean, True if user wants metric units
  • response: 200 | 500 server error
    • content-type: application/json
    • body: user's new weather settings
      • city: String
      • country: String
      • celsius: Boolean
